HTML

Hypertext Markup Language is the standard markup Language for creating web pages and web applications.

* h1-6는 헤딩 태그이며 중요도를 1-6으로 지정 가능
* strong은 문서에서 사용하는 진하게와 같음
* br은 엔터와 같은 줄바꿈 기능
* p는 하나의 단락을 지정하는 태그(닫기 필요)
* tap키는 들여쓰기 기능(언어 편집시 사용)
* ol(Ordered list)태그는 순서가 있는 목록
* ul(Unordered list)태그는 순서가 없는 목록
* li(list)태그는 ol, ul의 자식 태그로 항목을 지정
* meta 태그는 웹서버와 웹브라우저 간에 상호 교환되는 정보를 정의하는데 사용
* meta charset은 HTML문서의 문자 인코딩 방식을 지정
* head태그는 문서 속성을 정의, 지정하는 태그 title, meta, link 등
* body태그는 본문의 레이아웃, 모양을 지정하는 태그
* a태그는 링크를 거는 기능을 가짐 a href(hypertext reference) 새탭에서 열기 target="_blank"
